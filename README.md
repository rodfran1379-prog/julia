# JULIA Herencias y Sucesiones

Sitio web profesional para firma legal especializada en herencias y sucesiones en Nuevo León, México.

## 🚀 Tecnologías

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## 📦 Instalación

```bash
npm install
```

## 🏃 Desarrollo

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```

## 📁 Estructura

```
src/
  sections/     # Componentes de secciones
  components/   # Componentes reutilizables
  hooks/        # Custom hooks
  types/        # Tipos TypeScript
public/
  images/       # Imágenes del sitio
```
